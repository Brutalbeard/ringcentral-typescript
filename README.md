# RingCentral TypeScript SDK

This SDK is modelled after the [RingCentral.NET](https://github.com/ringcentral/ringcentral.net) SDK which is the most popular RingCentral SDK for static compiled languages.


## Compile

```
yarn watch
```


## Test

```
yarn test
```
